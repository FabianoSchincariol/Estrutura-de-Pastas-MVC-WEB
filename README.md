# Estrutura-de-Pastas-MVC-WEB
Estrutura Basica de pastas em MVC para web

### Comandos iniciais

#### Para criação do arquivo package.json
```bash
npm init
```

#### Para instalação dos modulos do NodeJS
```bash
npm install node-modules
```
